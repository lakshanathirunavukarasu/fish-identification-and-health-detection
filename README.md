# Fish Identification and Health Detection

A real-time AI-powered system designed to identify fish species and detect fish health conditions using neural networks and deep learning techniques. The project analyzes uploaded fish images, performs preprocessing and feature extraction, and predicts fish species and health status with high accuracy.

## Features
- Real-time fish species identification
- Fish health condition detection
- Image preprocessing and feature extraction
- Neural network based prediction model
- User-friendly interface using Streamlit
- FastAPI integration for real-time predictions
- Accurate and efficient results

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Pandas
- Streamlit
- FastAPI
- Deep Learning
- Machine Learning

## Project Workflow
1. Upload fish image
2. Image preprocessing
3. Feature extraction
4. Neural network analysis
5. Fish species prediction
6. Health status detection
7. Display prediction results
